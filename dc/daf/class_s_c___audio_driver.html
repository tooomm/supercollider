virtual bool SC_AudioDriver::DriverSetup ( int *  outNumSamplesPerCallback,
double *  outSampleRate 
)
protectedpure virtual

DriverSetup() should init the driver and write the num of samples per callback and the sample rate into the two addresses supplied as arguments. The driver will have access to the "preferred" values of these two args (mPreferredHardwareBufferFrameSize, mPreferredSampleRate) and ideally should follow them. This method should open the resources (and return true if successful), but shouldn't really start the streaming (this is the responsibility of DriverStart()).

Implemented in SC_PortAudioDriver, and SC_JackDriver.
