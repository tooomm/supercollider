Detailed Description

template<typename T, std::size_t bytes = 8 * 1024 * 1024>
class nova::tl_allocator< T, bytes >

realtime allocator: allocates from a thread-local memory pool via the malloc implementation of tlsf

it is not completely stl compliant memory allocated from one instance can be deallocated from each other instance of this class as long as it is done from the same thread
