Detailed Description

template<typename T, class Alloc = std::allocator<T>>
class nova::sized_array< T, Alloc >

dynamically sized array

array class, inspired by boost.array, where the size is specified during the time of the construction. in contrary to std::vector, it is guaranteed to use continuous memory. the memory is allocated in the constructor an freed in the destructor, these are the only places, where the allocator functions are called.
