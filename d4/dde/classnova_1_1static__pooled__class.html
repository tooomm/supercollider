Detailed Description

template<typename tag, std::size_t pool_size, bool pool_locking = false, unsigned int recover_count = 0>
class nova::static_pooled_class< tag, pool_size, pool_locking, recover_count >

base class for a class, which uses a static memory pool for memory allocation.

memory is allocated from a static pool and freed to a lock-free freelist, which is freed during the memory allocation routine. this way, the derived class can be allocated and freed from different threads, while all the access to the pool is done from the allocating thread.

Template Parameters
pool_lockingspecifies the locking policy for the object pool (default: nonblocking)
recover_countlimits the number of objects to be disposed from the freelist to avoid (default: dispose all objects of freelist)
